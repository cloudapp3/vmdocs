# Contributing

Thanks for helping improve the vminfo documentation site.

## Workflow

1. Fork or branch this repository.
2. Run `pnpm install`.
3. Start the vminfo docs locally with `pnpm docs:dev:vminfo`.
4. Make focused edits under `sites/vminfo/docs/`.
5. Verify with `pnpm docs:build:vminfo`.
6. Open a pull request.

## Notes

- Keep links valid and avoid broken image paths.
- Keep changes small and focused.
- Do not commit `node_modules/` or `sites/*/docs/.vitepress/dist/`.
