# vmbench 中文首页

vmbench 是一个跨平台 VPS / 主机测评工具，强调原始指标、结构化报告和对比分析，不输出 benchmark 综合总分、等级或 category score。

## 核心能力

- CLI / TUI 双入口
- JSON / HTML / Console 报告
- `compare` 报告对比
- VPS suite：hardware、route、ping、speed、ip_quality、mail、media
- MCP stdio server，供本地 LLM 客户端安全调用
