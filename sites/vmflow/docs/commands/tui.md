# vmflow tui

```bash
vmflow tui [-token TOKEN]
```

Alias: `vmflow t`.

Launches the terminal dashboard against the local daemon. See [TUI Dashboard](../guide/tui) for views and controls.

## Examples

```bash
vmflow tui
VMFLOW_CONTROL_TOKEN=change-me vmflow tui
```
