# vmflow tui

```bash
vmflow tui [-token TOKEN]
```

Alias: `vmflow t`.

Abre el panel de terminal del daemon local. Consulta [Panel TUI](./tui-guide).

## Ejemplos

```bash
vmflow tui
VMFLOW_CONTROL_TOKEN=change-me vmflow tui
```
