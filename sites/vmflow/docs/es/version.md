# vmflow version

```bash
vmflow version [-json]
```

Alias: `vmflow v`.

Imprime los metadatos de build: versión, commit y hora de build. Añade `-json` para una salida legible por máquina.

## Ejemplos

```bash
vmflow version
vmflow version -json
```

Útil para verificar una instalación o rellenar informes de fallos.
