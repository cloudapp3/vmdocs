# ローカル管理

サポートされる管理インターフェースは `vmflow ctl` と `vmflow tui` です。
デーモンはループバック専用の内部通信を使用します。この通信は公開連携 API
ではなく、外部向けの互換性は保証されません。

管理ポートは常に `127.0.0.1` にバインドされ、設定するのはローカルポート
だけです。

```yaml
control_port: 19090
```

状態、ルール、統計、事前チェック、リロードには組み込みコマンドを使用して
ください。[`vmflow ctl`](./ctl) と [`vmflow tui`](./tui) を参照してください。

## リモート管理

SSH でループバックポートを転送し、ローカルの CLI/TUI を使用します。

```bash
ssh -N -L 19090:127.0.0.1:19090 user@server
vmflow ctl rules
```
