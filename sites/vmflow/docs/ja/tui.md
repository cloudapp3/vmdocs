# vmflow tui

```bash
vmflow tui [-token TOKEN]
```

エイリアス: `vmflow t`。

ローカルデーモンのターミナルダッシュボードを起動します。ビューと操作については [TUI ダッシュボード](./tui-guide)を参照してください。

## 例

```bash
vmflow tui
VMFLOW_CONTROL_TOKEN=change-me vmflow tui
```
