# vmflow version

```bash
vmflow version [-json]
```

エイリアス: `vmflow v`。

ビルドメタデータ — バージョン、コミット、ビルド時刻 — を表示します。機械可読な出力には `-json` を追加してください。

## 例

```bash
vmflow version
vmflow version -json
```

インストールの確認やバグ報告の作成に役立ちます。
