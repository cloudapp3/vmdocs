# vmflow tui

```bash
vmflow tui [-token TOKEN]
```

Alias: `vmflow t`.

Abre o painel de terminal do daemon local. Consulte [Painel TUI](./tui-guide).

## Exemplos

```bash
vmflow tui
VMFLOW_CONTROL_TOKEN=change-me vmflow tui
```
