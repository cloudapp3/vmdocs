# vmflow version

```bash
vmflow version [-json]
```

Alias: `vmflow v`.

Exibe os metadados de build — versão, commit e horário de build. Adicione `-json` para uma saída legível por máquina.

## Exemplos

```bash
vmflow version
vmflow version -json
```

Útil para verificar uma instalação ou preencher relatórios de bugs.
