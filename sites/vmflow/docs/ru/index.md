## Установка одной командой

```bash
curl -fsSL https://raw.githubusercontent.com/cloudapp3/vmflow/main/install.sh | bash
```

Запустите демон:

```bash
vmflow daemon -config ./examples/config.yaml
```

Опросите его из другого терминала:

```bash
vmflow ctl health
vmflow ctl rules
vmflow ctl stats
vmflow ctl precheck
```

## Почему vmflow

vmflow создан для разработчиков и операторов, которым нужно лёгкое внутрипроцессное L4-перенаправление — как standalone-демон или как библиотека внутри более крупной плоскости управления.

Используйте его, когда вам нужно:

- перенаправлять трафик TCP/UDP между портами с ограничениями для каждого правила
- управлять правилами перенаправления из собственного желаемого состояния или базы данных
- проверять конфигурацию перенаправления перед применением
- предоставлять статистику правил и перезагрузку через небольшой локальный API
- встроить перенаправление в другой сервис на Go без добавления базы данных или веб-интерфейса
