# Предпроверка

Предпроверка валидирует конфигурацию **без её применения**. Она запускается автоматически перед каждой `reload`, а также её можно запустить по запросу, чтобы безопасно проверить изменения конфигурации.

## Запуск

```bash
vmflow ctl precheck
```

Выполните `vmflow ctl precheck` для проверки текущей конфигурации.

## Что проверяется

Предпроверка выдаёт список результатов, каждый из которых является либо **ошибкой**, либо **предупреждением**. Ошибки блокируют перезагрузку; предупреждения — нет.

| Проверка | Важность | Примечания |
| --- | --- | --- |
| Валидация модели правила | ошибка | Некорректные поля правила. |
| `duplicate_rule_id` | ошибка | Один и тот же ID встречается в снимке более одного раза. |
| Конфликт слушателей | ошибка | Два правила претендуют на один и тот же `listen_addr:port`. |
| Доступность порта | ошибка | Фактически пытается выполнить bind на порт прослушивания, чтобы убедиться в его доступности. |
| Разрешение DNS адресата | ошибка | `target_addr` должен разрешаться. |
| Конфигурация домена HTTPS | — | _Отключено в текущей сборке_ (протоколы http/https отклоняются). |
| Адрес ACME HTTP-01 | — | _Отключено в текущей сборке_ (подсистема ACME выключена). |
| Привилегированный порт | предупреждение | Привязка привилегированных портов (<1024) обычно требует повышенных прав. |

## Пример ответа

```json
{
  "config_path": "./examples/config.yaml",
  "rule_count": 1,
  "result": {
    "ok": true,
    "error_count": 0,
    "warning_count": 0,
    "checked_rules": 1,
    "checked_time_ms": 1,
    "items": []
  }
}
```

## Как перезагрузка это использует

`vmflow ctl reload` сначала запускает те же проверки. Если `error_count > 0`, перезагрузка отклоняется, а работающие правила остаются ровно в прежнем состоянии. Это делает правки конфигурации безопасными для применения через автоматизацию: повреждённая конфигурация не может примениться частично.

::: warning Локальные порты
Проба доступности порта кратко открывает локального слушателя. Если в вашем окружении запрещено создание локальных сокетов, предпроверка (и, следовательно, перезагрузка) не сможет полностью проверить доступность порта.
:::
