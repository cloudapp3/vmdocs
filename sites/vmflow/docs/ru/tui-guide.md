# Панель TUI

vmflow поставляется с терминальным интерфейсом для просмотра состояния правил и счётчиков трафика локального демона.

## Запуск

```bash
vmflow tui
```

Если аутентификация включена, передайте токен доступа:

```bash
vmflow tui -token <token>
# or
VMFLOW_CONTROL_TOKEN=<token> vmflow tui
```

## Режимы

Нажмите <kbd>Tab</kbd> для циклического переключения между режимами:

| Режим | Показывает |
| --- | --- |
| **Dashboard** | Общее состояние здоровья, количество работающих правил, время работы (uptime). |
| **Rules** | Список работающих правил с живыми счётчиками; поддерживает фильтрацию правил по имени. |
| **Detail** | Подробности о выбранном правиле. |

## Когда использовать

TUI — самый быстрый способ ответить на вопрос «что vmflow делает прямо сейчас?» без сбора метрик. Для долгосрочной истории направьте Prometheus на `/metrics` — TUI показывает только текущее состояние из памяти.
