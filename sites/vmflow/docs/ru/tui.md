# vmflow tui

```bash
vmflow tui [-token TOKEN]
```

Псевдоним: `vmflow t`.

Запускает терминальную панель локального демона. См. [Панель TUI](./tui-guide).

## Примеры

```bash
vmflow tui
VMFLOW_CONTROL_TOKEN=change-me vmflow tui
```
