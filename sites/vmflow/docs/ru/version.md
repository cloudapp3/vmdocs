# vmflow version

```bash
vmflow version [-json]
```

Псевдоним: `vmflow v`.

Выводит метаданные сборки — версию, коммит и время сборки. Добавьте `-json` для машиночитаемого вывода.

## Примеры

```bash
vmflow version
vmflow version -json
```

Полезно для проверки установки или оформления сообщений об ошибках.
