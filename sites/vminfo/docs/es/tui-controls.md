# Controles de la TUI

| Tecla | Acción |
| --- | --- |
| `q` / `ctrl+c` | Salir |
| `?` | Mostrar / ocultar ayuda |
| `p` | Pausar / reanudar |
| `+` / `-` | Ajustar intervalo |
| `r` | Actualizar ahora |
| `tab` | Cambiar entre vista general / procesos |
| `↑` / `↓` | Mover selección |
| `s` | Rotar orden |
| `t` | Vista en árbol |
| `/` | Filtrar procesos |
| `k` | Enviar SIGTERM al proceso seleccionado |
| `K` | Mostrar / ocultar hilos del kernel de Linux |
| `enter` / `y` | Confirmar eliminación |
| `esc` / `n` | Cancelar |

Insignias de estado usadas por la TUI:

- `LIVE`
- `PAUSED`
- `LOADING`
- `ERROR`
- `STALE`

El modo TUI requiere una TTY real.
