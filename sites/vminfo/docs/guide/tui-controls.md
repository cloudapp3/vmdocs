# TUI Controls

| Key | Action |
| --- | --- |
| `q` / `ctrl+c` | Quit |
| `?` | Toggle help |
| `p` | Pause / resume |
| `+` / `-` | Adjust interval |
| `r` | Refresh now |
| `tab` | Switch overview / processes |
| `↑` / `↓` | Move selection |
| `s` | Cycle sort |
| `t` | Tree view |
| `/` | Filter processes |
| `k` | SIGTERM selected process |
| `K` | Show / hide Linux kernel threads |
| `enter` / `y` | Confirm kill |
| `esc` / `n` | Cancel |

Status badges used by the TUI:

- `LIVE`
- `PAUSED`
- `LOADING`
- `ERROR`
- `STALE`

TUI mode requires a real TTY.
