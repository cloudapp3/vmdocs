# TUI 操作

| キー | アクション |
| --- | --- |
| `q` / `ctrl+c` | 終了 |
| `?` | ヘルプの表示切替 |
| `p` | 一時停止 / 再開 |
| `+` / `-` | 更新間隔の調整 |
| `r` | 今すぐ更新 |
| `tab` | 概要 / プロセスの切り替え |
| `↑` / `↓` | 選択を移動 |
| `s` | ソート順を切り替え |
| `t` | ツリー表示 |
| `/` | プロセスをフィルター |
| `k` | 選択したプロセスに SIGTERM を送信 |
| `K` | Linux カーネルスレッドの表示 / 非表示 |
| `enter` / `y` | kill を確定 |
| `esc` / `n` | キャンセル |

TUI で使われるステータスバッジ：

- `LIVE`
- `PAUSED`
- `LOADING`
- `ERROR`
- `STALE`

TUI モードは実際の TTY が必要です。
