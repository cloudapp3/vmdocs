# TUI 컨트롤

| 키 | 동작 |
| --- | --- |
| `q` / `ctrl+c` | 종료 |
| `?` | 도움말 표시/숨기기 |
| `p` | 일시정지 / 재개 |
| `+` / `-` | 간격 조정 |
| `r` | 지금 새로고침 |
| `tab` | 개요 / 프로세스 전환 |
| `↑` / `↓` | 선택 이동 |
| `s` | 정렬 순환 |
| `t` | 트리 보기 |
| `/` | 프로세스 필터 |
| `k` | 선택한 프로세스에 SIGTERM 전송 |
| `K` | Linux 커널 스레드 표시/숨기기 |
| `enter` / `y` | 종료 확인 |
| `esc` / `n` | 취소 |

TUI에서 사용하는 상태 배지:

- `LIVE`
- `PAUSED`
- `LOADING`
- `ERROR`
- `STALE`

TUI 모드는 실제 TTY가 필요합니다.
