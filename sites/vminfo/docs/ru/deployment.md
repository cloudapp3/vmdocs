# Развёртывание

Этот сайт документации рассчитан на чистое развёртывание на Cloudflare Pages из монорепозитория документации `cloudapp3/vmdocs`.

## Настройки Cloudflare Pages

| Настройка | Значение |
| --- | --- |
| Предустановка фреймворка | VitePress или None |
| Команда сборки | `pnpm docs:build:vminfo` |
| Каталог вывода сборки | `sites/vminfo/docs/.vitepress/dist` |
| Корневой каталог | `/` |
| Версия Node.js | 20 или новее |

Если вашему проекту Pages нужно устанавливать зависимости во время сборки, используйте:

```bash
corepack enable && pnpm install --frozen-lockfile && pnpm docs:build:vminfo
```

## Пользовательский домен

- Используйте поддомен для документации, например `vminfo.bestcheapvps.org`
- Сначала добавьте пользовательский домен в проекте Cloudflare Pages
- Затем настройте DNS, как указывает Cloudflare
- Не создавайте только ручной CNAME без привязки проекта Pages

## Базовый путь

- Для пользовательского домена в корне сайта не задавайте `base`
- Для развёртывания в подпути, например `https://example.com/vminfo/`, задайте `base: "/vminfo/"`

## Вывод

Cloudflare Pages должен публиковать статический сайт из `sites/vminfo/docs/.vitepress/dist`.
