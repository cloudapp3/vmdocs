# `vminfo kill`

Отправляет `SIGTERM` процессу Linux.

::: warning
Эта команда завершает процесс. Перед запуском убедитесь, что PID указан верно.
:::

## Использование

```bash
vminfo kill 1234
```

## Примечания

- Только Linux
- на сборках не под Linux возвращает неподдерживаемые заглушки
- может потребовать прав, если целевой процесс принадлежит другому пользователю
