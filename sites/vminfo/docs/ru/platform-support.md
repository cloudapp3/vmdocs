# Поддержка платформ

| Возможность | Linux | macOS | Windows |
| --- | --- | --- | --- |
| `summary` / `watch` | ✅ | ✅ | ✅ |
| TUI | ✅ | ✅ | ✅ |
| Веб-дашборд | ✅ | ✅ | ✅ |
| `ps` / `kill` | ✅ | ⚠️ заглушка | ⚠️ заглушка |
| `update --check` | ✅ | ✅ | ✅ |
| `update` установка | ✅ | ✅ | ⚠️ только проверка |

Примечания:

- TUI требует реальный TTY.
- `ps` и `kill` по дизайну доступны только на Linux.
- Сборки не для Linux сохраняют неподдерживаемые заглушки для функций работы с процессами.
- Windows может проверять обновления, но самозамена не поддерживается.
