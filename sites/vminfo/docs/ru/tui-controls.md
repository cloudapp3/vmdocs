# Управление TUI

| Клавиша | Действие |
| --- | --- |
| `q` / `ctrl+c` | Выход |
| `?` | Показать/скрыть справку |
| `p` | Пауза / возобновление |
| `+` / `-` | Изменить интервал |
| `r` | Обновить сейчас |
| `tab` | Переключить обзор / процессы |
| `↑` / `↓` | Переместить выделение |
| `s` | Циклически сменить сортировку |
| `t` | Древовидный вид |
| `/` | Фильтр процессов |
| `k` | SIGTERM выбранного процесса |
| `K` | Показать / скрыть потоки ядра Linux |
| `enter` / `y` | Подтвердить завершение |
| `esc` / `n` | Отмена |

Бейджи статуса, используемые в TUI:

- `LIVE`
- `PAUSED`
- `LOADING`
- `ERROR`
- `STALE`

Режим TUI требует реальный TTY.
