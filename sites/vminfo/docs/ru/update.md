# `vminfo update`

Проверяет GitHub Releases и при возможности устанавливает последнюю тегированную сборку.

## Использование

```bash
vminfo update
vminfo update --check
vminfo update --version v0.1.0
```

## Примечания

- `--check` только проверяет наличие обновлений.
- `--version` проверяет или устанавливает конкретный тег релиза.
- Dev-сборка не может обновить сама себя без `--version`.
- На Windows режим установки не поддерживается, так как бинарный файл не может заменить сам себя на месте.

## Пример

```bash
vminfo update --check
```
