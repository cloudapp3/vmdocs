# Веб-дашборд

`vminfo --web` запускает легковесные HTTP API и дашборд только для чтения с переключаемыми темами.

## Запуск сервера

```bash
vminfo --web
```

Адрес по умолчанию:

```text
http://127.0.0.1:20021
```

Произвольный адрес:

```bash
vminfo --web --bind 0.0.0.0 --port 8080 --interval 1s
```

Также можно запустить TUI вместе с дашбордом:

```bash
vminfo --web --tui
```

## Авторизация

По умолчанию дашборд и API доступны локально и без авторизации.

При включённом `--token`:

```bash
vminfo --web --token
vminfo --web --token my-secret
```

- пустой `--token` автоматически генерирует URL-безопасный токен
- `--token my-secret` использует фиксированный токен
- при первом успешном заходе через `/?token=...` устанавливается cookie для последующих запросов
- `/healthz` остаётся открытым для локальных проверок
- `/`, `/api/v1/*` и `/ws` требуют токен или cookie авторизации
- режим с защитой токеном не открывает разрешающий `Access-Control-Allow-Origin: *`
- поднятие WebSocket требует, чтобы origin браузера совпадал с хостом дашборда

::: warning Удалённый доступ
При привязке к `0.0.0.0` включайте `--token`, если только дашборд не открыт исключительно для доверенной частной сети.
:::

## Эндпоинты

| Эндпоинт | Назначение |
| --- | --- |
| `GET /healthz` | Открытая проверка здоровья |
| `GET /api/v1/snapshot` | Полный снимок рантайма |
| `GET /api/v1/cpu` | Данные CPU |
| `GET /api/v1/memory` | Данные памяти и swap |
| `GET /api/v1/disk` | Данные файловой системы и дискового ввода-вывода |
| `GET /api/v1/network` | Итоги по сети и данные интерфейсов |
| `GET /api/v1/processes` | Список процессов |
| `GET /api/v1/system` | Метаданные хоста |
| `GET /api/v1/health` | Оценка здоровья и предупреждения |
| `POST /api/v1/net/diag` | Запуск сетевой диагностики (dns / port / ping / ip) |
| `GET /ws` | Живой поток снимков |

Полные примеры тел запросов и параметры запроса см. в [справочнике HTTP API](/ru/api).

## Темы

В дашборд встроены переключаемые темы из шапки: **Auto**, **Neon**, **Light**, **Terminal** и **Synthwave**. «Auto» повторяет цветовую схему ОС.

Шрифт [JetBrains Mono](https://www.jetbrains.com/lp/mono/) **встроен** в бинарник, поэтому дашборд полностью самодостаточен — он рендерится задуманным моноширинным шрифтом и работает офлайн без внешних запросов шрифтов.
