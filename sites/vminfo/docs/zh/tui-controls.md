# TUI 控制

| 按键 | 动作 |
| --- | --- |
| `q` / `ctrl+c` | 退出 |
| `?` | 切换帮助 |
| `p` | 暂停 / 恢复 |
| `+` / `-` | 调整刷新间隔 |
| `r` | 立即刷新 |
| `tab` | 切换概览 / 进程 |
| `↑` / `↓` | 移动选中项 |
| `s` | 循环切换排序 |
| `t` | 树形视图 |
| `/` | 过滤进程 |
| `k` | 向选中进程发送 SIGTERM |
| `K` | 显示 / 隐藏 Linux 内核线程 |
| `enter` / `y` | 确认终止 |
| `esc` / `n` | 取消 |

TUI 使用的状态标识：

- `LIVE`
- `PAUSED`
- `LOADING`
- `ERROR`
- `STALE`

TUI 模式需要真实的 TTY。
